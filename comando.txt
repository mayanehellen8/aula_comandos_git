git (comando para acessar o git pelo terminal)
git --version (pra saber a versão do commit)
git init (abrir/iniciar o git)
git config (configurar o git)
git config user.name (configurar/saber o nome)
git config user.email (saber/configurar email)
git branch -M main (mudar o nome da branch principal pra main)
git config --global (usar o mesmo email associado ao criar novos repositórios)
git status (mostra o status do git, o que está acontecendo, sendo criado ou iniciado)
git add . (para tornar todos os arquivos parte do repositório)
git add (nome do arquivo) = (tornar apenas este arquivo parte do repositório)
git commit -m "add new file: (nome do arquivo)" = (salvar no histórico
dar um save para não perder depois e sempre poder acessar)
git log (mostrar histórico de commit)
git checkout -b nome_da_branch (espécie de nova sessão, cria e acessa outra branch)
git checkout (ou switcht) nome_da_branch (para apenas trocar de branch)
git merge nome_da_branch (para salvar informações da (s) outra branch (s)
 acessada/alterada, na branch atual)
 git remote remove origin (resolver o problema de a branch estar vinculada a uma origem que você não quer)

3 comandos para adicionar branch ao github: 

1 - git remote add origin https://github.com/mayanehellen8/aula1_git.git (estou adicionando ao github com os
 comandos e dando o site em que os arquivos serão enviados)
2 - git branch -M main (escolhendo a branch)
3 - git push -u origin main (push... passo final para adicionar ao github)
